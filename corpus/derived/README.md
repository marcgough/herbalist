# Derived Knowledge

This folder holds structured outputs extracted from the corpus, for example:

- edition families
- evidence tables and source-linked term graphs
- entities
- claims
- cautions
- preparations
- plant parts
- synonyms
- later semantic embeddings

Everything here must remain traceable back to source passages.

Current live layers:

- `edition-families/`
- `evidence/`
- `term-families/`
- `seed-catalog/`
